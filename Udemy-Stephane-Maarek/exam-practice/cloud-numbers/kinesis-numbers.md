## Kinesis Data Stream
* Retention up to _365 days_  

__Kinesis Data Stres - Provisioned Mode__  
* Each shard gets 1MB/s in (or 1000 records per second)
* Each shard gets 2MB/s out

__Kinesis Data Stres - On-demand  Mode__  
* Default capacity provisioned (4 MB/s in or 4000 records per second)
* Default capacity provisioned (4 MB/s in or 4000 records per second)

## Kinesis Data Firehose
* Kinesis Data Firehose is the easiest way to load streaming data into data stores and analytics tools.
* Firehose Destinations include: Amazon S3, Amazon Redshift, Amazon Elasticsearch Service, and Splunk

__Firehose Buffer Sizging__  
* Minimum Buffer Time is _1 Minute_
